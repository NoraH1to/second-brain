---
html
```html
<div class="rounded-wrapper">
  <div class="content">
  </div>
</div>
```

css
```css
.rounded-wrapper {
  overflow: hidden;
  border-radius: 50%;
  height: 200px;
  width: 200px;
}
.content {
  transform: translatez(0);
  height: 100%;
  width: 150%;
  background: gray;
}

```
